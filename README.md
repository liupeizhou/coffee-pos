# 咖啡店点单系统 (Coffee POS)

一个基于 Electron + React 的咖啡店点单和结算系统。

## 功能特点

- **产品管理** - 管理分类、产品、图片和做法选项
- **订单管理** - 快速下单、多种支付方式、支持会员折扣
- **员工管理** - 工号登录、权限管理（管理员/员工）
- **换班系统** - 早中晚班管理、交接结账
- **报表系统** - 产品销量分析、每日营收、同比环比、Excel导出
- **数据管理** - 支持清除运营数据（管理员操作）

## 技术栈

- **前端**: React 18 + TypeScript
- **桌面框架**: Electron
- **数据库**: SQLite (sql.js)
- **状态管理**: React Context
- **Excel导出**: XLSX

## 开发

```bash
# 安装依赖
npm install

# 开发模式
npm run start

# 构建
npm run build
```

## 使用说明

1. 首次启动需要创建管理员账号
2. 员工登录使用工号和密码
3. 下班前需要交班结算
4. 管理员可查看报表和导出数据
